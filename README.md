# 简易天气预报 Simple Weather Forcast
荣获：2016厦门智游网安科技有限公司"智游杯"HTML5项目团队一等奖
## 特色
- 基于百度气象API
- 可以分辨用户设备，自动识别是手机端还是电脑端
- 气象温度曲线
- 生活小贴士

[欢迎访问](http://139.199.67.181:8085)
## 所用技术
- HTML5
- CSS3
- JQuery
- PHP+MySQL
